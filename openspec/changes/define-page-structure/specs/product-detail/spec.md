## MODIFIED Requirements

### Requirement: 商品詳細情報を表示する
システムは `/products/[slug]` で対象商品の名前・ブランド・価格・カテゴリ・画像・説明文を表示しなければならない (SHALL)。
期間限定商品（`is_seasonal=true`）には「期間限定」バッジを表示する。

#### Scenario: ユーザーが商品詳細ページを開く
- **WHEN** ユーザーが `/products/[slug]` にアクセスする
- **THEN** システムは該当商品の名前・ブランド・価格・カテゴリ・画像・説明文を表示する

#### Scenario: 期間限定商品の場合
- **WHEN** `is_seasonal=true` の商品詳細ページを開く
- **THEN** システムは商品情報に「期間限定」バッジを表示する

#### Scenario: 存在しない商品 slug にアクセスする
- **WHEN** 存在しない slug で `/products/[slug]` にアクセスする
- **THEN** システムは 404 ページを返す

### Requirement: 商品詳細ページの OGP を設定する
システムは各商品詳細ページに商品名を含む `<title>` と商品画像を使った OGP タグを出力しなければならない (SHALL)。

#### Scenario: SNS でシェアする
- **WHEN** `/products/[slug]` の URL を SNS にシェアする
- **THEN** SNS カードは商品名・商品画像・説明文を表示する

## ADDED Requirements

### Requirement: 購入リンクを表示する
システムは商品詳細ページに外部の購入先へのリンクボタンを表示しなければならない (SHALL)。
購入リンクが未設定の商品では購入リンクを表示しない。

#### Scenario: 購入リンクがある商品を表示する
- **WHEN** `purchase_url` フィールドが設定された商品の詳細ページを開く
- **THEN** システムは「購入する」リンクボタンを表示し、クリックすると新規タブで購入先 URL を開く

#### Scenario: 購入リンクがない商品を表示する
- **WHEN** `purchase_url` フィールドが空の商品の詳細ページを開く
- **THEN** システムは購入リンクボタンを表示しない

### Requirement: 関連商品を表示する
システムは商品詳細ページの末尾に同カテゴリの他商品を最大3件関連商品として表示しなければならない (SHALL)。
関連商品から現在表示中の商品は除外する。

#### Scenario: 同カテゴリの商品が3件以上ある場合
- **WHEN** 同カテゴリに3件以上の他商品がある商品詳細ページを開く
- **THEN** システムは同カテゴリの商品を3件表示する

#### Scenario: 同カテゴリの商品が自身のみの場合
- **WHEN** 同カテゴリに他商品が存在しない商品詳細ページを開く
- **THEN** システムは関連商品セクションを表示しない
