## ADDED Requirements

### Requirement: ヘッダーにロゴとナビゲーションを表示する
システムはすべてのページでヘッダーを表示しなければならない (SHALL)。
ヘッダーは左側にロゴ、右側にナビゲーションリンク（商品一覧・カテゴリ・このサイトについて）を配置する。

#### Scenario: PC でヘッダーを表示する
- **WHEN** ユーザーが 1024px 以上のデバイスで任意のページにアクセスする
- **THEN** システムはヘッダー左にロゴ、右に「商品一覧」「カテゴリ」「このサイトについて」のナビリンクを水平表示する

### Requirement: SP でハンバーガーメニューを表示する
システムは 639px 以下の画面幅でナビゲーションをハンバーガーメニューに切り替えなければならない (SHALL)。
ハンバーガーアイコンをタップするとドロワーメニューが開き、全ナビリンクを縦に表示する。

#### Scenario: SP でハンバーガーメニューを開く
- **WHEN** ユーザーが 639px 以下のデバイスでハンバーガーアイコンをタップする
- **THEN** システムはナビゲーションリンクをドロワー形式で表示する

#### Scenario: SP でドロワーメニューを閉じる
- **WHEN** ドロワーが開いている状態でユーザーがいずれかのナビリンクをタップする
- **THEN** システムはドロワーを閉じて対象ページに遷移する

### Requirement: フッターにロゴ・リンク・コピーライトを表示する
システムはすべてのページでフッターを表示しなければならない (SHALL)。
フッターにはロゴ・主要ナビリンク・コピーライト表記を含む。

#### Scenario: フッターが表示される
- **WHEN** ユーザーが任意のページ最下部にスクロールする
- **THEN** システムはロゴ・ナビリンク・「© ChocoMint Sweets Guide」などのコピーライトを表示する

### Requirement: パンくずリストをトップページ以外で表示する
システムはトップページ（`/`）を除くすべてのページでパンくずリストを表示しなければならない (SHALL)。
パンくずは「ホーム > 現在のページ名」形式で、各段階がリンクになっている。

#### Scenario: 商品一覧ページのパンくずリスト
- **WHEN** ユーザーが `/products` にアクセスする
- **THEN** システムは「ホーム > 商品一覧」のパンくずリストを表示し、「ホーム」はリンクになっている

#### Scenario: 商品詳細ページのパンくずリスト
- **WHEN** ユーザーが `/products/[slug]` にアクセスする
- **THEN** システムは「ホーム > 商品一覧 > 商品名」のパンくずリストを表示する

#### Scenario: トップページはパンくずを非表示にする
- **WHEN** ユーザーが `/` にアクセスする
- **THEN** システムはパンくずリストを表示しない
